# CPU Usage

這個資料夾已簡化為單一正確版本：`CpuUsageWidget.Wpf`

## 啟動

直接雙擊根目錄的 `run-widget.bat`

它會自動進入 `CpuUsageWidget.Wpf` 並執行：

```powershell
dotnet run
```

## 主要檔案

- `CpuUsageWidget.Wpf/MainWindow.xaml`
- `CpuUsageWidget.Wpf/MainWindow.xaml.cs`
- `CpuUsageWidget.Wpf/settings.txt`
